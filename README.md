# Telegram Bot для учета показателей сотрудников

## Описание
Бот для сбора и анализа показателей сотрудников с системой грейдов (8, 9, 10).

## Функции
- Регистрация с паролем `903829`
- Выбор грейда при регистрации
- Ежедневное внесение показателей
- Автоматический расчет недельных и квартальных сумм
- Динамический план (пропорционально времени квартала)
- Отчеты: личные, общие, по сотрудникам
- Уведомления всем пользователям
- Управление показателями и планами

## Установка
```bash
pip install -r requirements.txt
python bot.py
